# ProductsAPI

Este é um projeto de estudos sobre Spring Boot e outros conceitos mais técnicos como:

- APIs Restful e APIs Gateway
- Microserviços
- Ferramentas de Mensageria
- TDD
- Autenticação JWT

## Tecnologias usadas

- Java 17 + Spring Boot
- RabbitMQ + Celery
- MongoDB e MSSQL
- Docker
